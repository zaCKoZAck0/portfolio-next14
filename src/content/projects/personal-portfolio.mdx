## Personal Portfolio

Developed a personal portfolio and blog website using Next.js 14, Tailwind CSS, Framer Motion and shadcn/ui. The website is hosted on Vercel and the blog posts are fetched from the Contentful CMS.

- **Live URL:** [zackozack.xyz](https://www.zackozack.xyz/)
- **GitHub Repository:** [zaCKoZAck0/portfolio-next14](https://github.com/zaCKoZAck0/portfolio-next14)

### Features

1. Responsive design
2. Animations using 'Framer Motion'
3. Dynamic content using Contentful (Projects, Profile, Blogs, etc.), yes its all MDX
4. GitHub and Leetcode API integration
5. MDX Blog posts
6. SEO Optimized Pages and Blogs
7. OG Image support
